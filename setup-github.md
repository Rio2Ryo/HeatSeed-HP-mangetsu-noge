# GitHubリポジトリのセットアップ手順

## 1. GitHubでリポジトリを作成

以下の名前でGitHubにリポジトリを作成してください：
**HeatSeed-HP-mangetsu-noge**

## 2. リモートを追加してプッシュ

```bash
git remote add origin https://github.com/YOUR_USERNAME/HeatSeed-HP-mangetsu-noge.git
git branch -M main
git push -u origin main
```

YOUR_USERNAMEをあなたのGitHubユーザー名に置き換えてください。
